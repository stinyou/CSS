Generally, you want to use a background image that does not affect text readability.































































The background image repeats to cover the section by default.
If you want the image to only repeat horizontally or vertically, the respective options are:
background-repeat: repeat-x;
OR
background-repeat: repeat-y;

Can also use no-repeat to make sure the image does not repeat horizontally or vertically. Can use background-position to set the position of the background. For example, the bottom right of the image.
